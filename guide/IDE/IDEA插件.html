                                                                                                                                                      • 要为活动元素生成 javadocs，请按 shift + alt + G。
                                                                                                                                                      • 要为当前 java 文件中的所有元素生成 javadocs，请按 shift + ctrl + alt + G。
                                                                                                                                                      • 删除当前/选定元素上的 javadocs 请按 shift + alt + Z。
                                                                                                                                                      • 删除当前类所有元素上的 javadocs：请按 shift + ctrl + alt + Z。
